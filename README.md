# Hangman Game

This project contains a simple Python Hangman game in `test.py`.

## How to run

### Option 1: Use VS Code Run and Debug
1. Open this folder in VS Code.
2. Make sure the Python extension is installed.
3. Select the Python interpreter at the bottom right.
4. Open `test.py`.
5. Open the Run and Debug view (`Ctrl+Shift+D`).
6. Choose `Python: Run test.py` and press `F5` or click the green run button.

### Option 2: Use the terminal
Open the integrated terminal in VS Code and run:

```powershell
cd "c:\python internship"
& .\.venv\Scripts\python.exe .\test.py
```

## Notes
- The file `test.py` is already complete and ready to run.
- If you get `python not found`, select the correct interpreter in VS Code or use the local virtual environment path above.
- After the game ends, type `yes` or `no` when asked whether to play again.
